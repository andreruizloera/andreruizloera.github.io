What the good free templates have in common

How to Design Better with AI

I looked at all the free templates on one of the big motion-design sites, expecting a bunch of different ideas. There’s really only one. The good ones aren’t good because of the animation. They’re good because each picks a specific fake company and stays consistent with it. That’s the post. The demos below are live, and I built them from scratch.

June 20, 2026 · 9 min read Design AnalysisCoherenceClaude

The free tier is usually filler, and plenty of these are. But the ones that look good all share something, and it took me a second pass to see what it was.

None of them is “a hero section.” It’s a hero for SkyElite Private Jets. For Orbis, an NFT drop. For Securify, Neuralyn, Datacore. There’s even a 404 page with its own fake company behind it. None of these businesses are real — someone made one up and designed for that exact thing instead of for “a website.”

What they all do

Across the good ones, the same five choices show up:

  1. Make up a company that sounds real.
  2. Pick a field that suits a dark, moody look — software, AI, crypto, luxury. Not a dentist.
  3. Build the page around one thing — a product shot, a 3D object, a dashboard.
  4. Use one slow background motion, and nothing else.
  5. One headline, one button.

It’s not the animation. It’s that everything on the page — the name, the colors, the type, the one moving thing — agrees with the same made-up company. Nothing on the page fights anything else. That consistency is what makes it look expensive, more than any single effect does.

The names

These are real template names, scrolling past:

Aetheris, Celestia, Luminex, Neuralyn, Nexora. They’re built the same way — a vaguely Latin or space-ish root with a tech-sounding suffix — and they all sound like a company that raised money. They don’t mean anything, which actually helps: a name that means nothing can’t clash with the rest of the design. The one forgettable template in the set is called “Modern Agency.”

The subjects

The subjects are telling too. It’s always software, AI, security, crypto, jets, the occasional museum. Never a plumber, never a dentist. Those don’t suit a black background and a slow gradient, so they’re just not in the set. The templates only take on subjects that already fit the look they wanted to use.

The motion

This is the part people get wrong when they copy the style. The free animated backgrounds are all the same kind of thing: slow, drifting, ambient. The names give it away — Fluid Motion, Aurora Waves, Silk Ripple. None of them is busy. The motion is there to set a mood, not to perform.

Using one instead of ten is the hard part, and it’s usually where my own attempts went wrong. Here are the actual effects, live on this page and written by hand. They’re all simple. That’s sort of the point.

The gradient at the top of the page is four colors and one keyframe. People assume it’s a shader. It isn’t: 

@keyframes gradient-drift {
  0%   { background-position: 0% 50%; }
  50%  { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

Things fade up as they come into view. It’s an IntersectionObserver and one CSS class. Most of the “cinematic” sites you’ve seen are mostly this: 

const io = new IntersectionObserver((entries) => {
  for (const e of entries) {
    if (e.isIntersecting) { e.target.classList.add('in'); io.unobserve(e.target); }
  }
}, { threshold: 0.2 });
document.querySelectorAll('.reveal').forEach((el) => io.observe(el));

The part that matters is the easing. That cubic-bezier(0.22, 1, 0.36, 1) — quick to start, slow to land — is most of the difference between this looking deliberate and looking like a slideshow. Default linear easing is usually the giveaway.

The off version isn’t broken. The text is fine and the layout is fine. It just didn’t commit to anything — a vague headline, three buttons instead of one, a placeholder where a real idea should be. That’s usually the actual problem, not a lack of effort. 

Build ONE hero section for Orbis — a digital collectibles label that does
single timed drops. Five rules, no exceptions:

1. Brand: scarcity as luxury. It should feel like you are already late.
2. The vertical earns the look: collectibles, so commit to near-black, one
   electric accent, and a single hero object that looks worth owning.
3. One artifact: a 3D collectible, slowly rotating and bobbing, lit so the
   edges catch. No grids of thumbnails, no logo wall.
4. One motion: the object rotates; nothing else moves. Respect
   prefers-reduced-motion and let it sit still if asked.
5. One message, one button: a countdown-flavored headline and a single
   "Join the drop" CTA. No secondary links begging for attention.

Type: one confident display face, one clean sans. Numbers should feel minted.
Rule of thumb: if a choice doesn't make the drop feel scarcer, cut it.
Constraints over adjectives.

Most of that prompt is about leaving things out: one object, one motion, one button. It’s the same five choices, written as instructions.

Why it works

You can’t really picture “a SaaS.” You can picture a specific black-and-gold jet site. A specific fake company gives the viewer something concrete, so the page reads as a real product instead of a placeholder. And it’s not much extra work — once everything lines up, it just looks intentional.

Doing this with AI

This is also why sites you build with AI tend to look generic. “Make it pop, add some animations” gives the model nothing to be consistent about. When I want something that doesn’t look like every other gradient-with-a-button, I give it the same five things:

  • Give it the fake company first. “A hero for SkyElite, a private jet charter for people who hate airports” gets you a real design. “A SaaS hero” gets you nothing.
  • Name one motion and rule out the rest. “One slow gradient in the background, nothing else moving” works better than “make it dynamic.”
  • Ask for one main thing and one button. A single thing to show, a single thing to do.
  • Mention reduced motion. One prefers-reduced-motion query turns the animation off for people who need it. Easy to forget, worth not forgetting.
  • Write the code yourself. Learn the technique; don’t copy someone’s actual template. Everything here is built from scratch, which I’ll admit is a funny thing to say after a whole post spent studying other people’s.

The actual takeaway

The good template wasn’t more talented than you. It just picked one specific idea and stuck to it.

Pick a specific fake company, design for it and nothing else, use one bit of motion, and say one thing. That’s most of the difference between looking generic and looking finished. The tools are all free. The choosing is the hard part.

So before you start, make up the specific company — SkyElite, Orbis, whatever — and build for that one thing. It’s a small change, and it does most of the work.
